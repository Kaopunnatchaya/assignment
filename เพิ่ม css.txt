เพื่อเพิ่ม CSS จาก Bootstrap ในเว็บแอปพลิเคชัน Flask ของคุณ เราสามารถทำได้โดยการเพิ่มลิงค์ไปยังไฟล์ CSS ของ Bootstrap ในไฟล์ HTML ของเรา นี่คือตัวอย่างการแก้ไขไฟล์ HTML เพื่อรวม Bootstrap:

1. **เพิ่ม Bootstrap CDN ในไฟล์ HTML**:

   - **`index.html`**:
     ```html
     <!doctype html>
     <html>
     <head>
         <title>ATM Application</title>
         <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
     </head>
     <body>
         <div class="container">
             <h1 class="mt-5">Welcome to the ATM</h1>
             <ul class="list-group mt-3">
                 <li class="list-group-item"><a href="{{ url_for('create_account') }}">Create Account</a></li>
                 <li class="list-group-item"><a href="{{ url_for('view_balance') }}">View Balance</a></li>
                 <li class="list-group-item"><a href="{{ url_for('deposit') }}">Deposit</a></li>
                 <li class="list-group-item"><a href="{{ url_for('withdraw') }}">Withdraw</a></li>
                 <li class="list-group-item"><a href="{{ url_for('delete_account') }}">Delete Account</a></li>
             </ul>
         </div>
         <script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
         <script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.9.2/dist/umd/popper.min.js"></script>
         <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
     </body>
     </html>
     ```

   - **`create_account.html`**:
     ```html
     <!doctype html>

                     <input type="text" name="account_number" class="form-control" id="account_number" required>
                 </div>
                 <div class="form-group">
                     <label for="username">Username:</label>
                     <input type="text" name="username" class="form-control" id="username" required>
                 </div>

                     <input type="number" step="0.01" name="balance" class="form-control" id="balance" required>
                 </div>

             </form>
             <a href="{{ url_for('index') }}" class="btn btn-secondary mt-3">Back</a>
     ```
